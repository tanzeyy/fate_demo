### 模型训练

- 请求地址：`http://192.168.25.105:5000/model/train`

- 请求方式：`POST`

- 请求参数：

  ```json
  {
      "user_id": 4, 
      "model_type": "lr",
      "model_params": {
        "penalty": "L2",
        "optimizer": "sgd",
        "eps": 1e-4,
        "alpha": 0.01,
        "max_iter": 10,
        "converge_func": "diff",
        "batch_size": 500,
        "learning_rate": 0.15,
        "decay": 1,
        "decay_sqrt": true,
        "init_param": {
          "init_method": "zeros"
        },
        "encrypt_param": {
          "method": "Paillier"
        },
        "cv_param": {
          "need_cv": false
        }
      },
      "party_id": [2,3],
      "data_info": {
        "4": "SELECT * FROM data.breast_homo_guest;",
        "2": "SELECT * FROM data.breast_homo_host;",
        "3": "SELECT * FROM data.breast_homo_host;"
      },
      "unique_id": "id",
      "attributes": ["x1", "x2", "x3", "x4"],
      "label_name" : "y"
  }
  ```

- 请求成功返回：

  ```json

  ```
